# **App Name**: Field Focus

## Core Features:

- Data Ingestion: Parse data from SportsCode XML and CSV formats, extracting relevant match events and game states.
- Tactical Metrics Computation: Calculate key tactical metrics, such as Seconds Per Press (SPP), Build25 Ratio, and Circle Entry Efficiency, providing a quantitative overview of team performance.
- Dynamic Zone Heatmap Generation: Visualize turnover frequency across different zones of the hockey pitch, using dynamic heatmaps for quick identification of critical areas.
- Dynamic Pressure Battle Chart: Generate a line chart comparing SPP values of opposing teams, dynamically shaded to indicate pressure dominance, offering insights into tactical advantages.
- Circle Entry Analysis Display: Visualize circle entry efficiency via a half-field diagram, displaying entries, shots, and efficiency percentages for each entry channel (Left, Center, Right).
- Automated Reporting: Produce A4-printable PDF reports with key metrics and visualizations, optimized for clarity and impact.

## Style Guidelines:

- Primary color: Deep blue (#194569), evoking the focus and strategy inherent in field hockey analytics.
- Background color: Light gray (#F0F4F7), creating a clean and professional backdrop that ensures readability and focus on the data.
- Accent color: Mustard yellow (#D4AC0D), employed sparingly to highlight key metrics and calls to action, drawing attention without overwhelming.
- Body and headline font: 'Inter', a sans-serif typeface, chosen for its clean, modern, machined, objective and neutral look.
- Crisp, minimalist icons to represent data types and actions. Ensure consistency and clarity.
- Responsive layout optimized for both screen viewing and A4 printing, ensuring seamless data consumption across different mediums.
- Subtle transitions for loading states and interactions. Animations should serve to clarify the data flow and enhance the user experience, not distract.
